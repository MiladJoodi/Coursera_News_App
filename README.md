This project contains a fully responsive navbar and a inspiring mobile menu.
I used soothing colors. For authentication I used Clerk, it help me to implement the user login feature in a very professional way.
It was fantastic. üëè
 I used 'Api Routes' in this project. The main page posts are loaded from a MongoDB database and I connected to it using Mongoose. After login with Email and verify code or google account and github you can create a new post in home page. üí™
